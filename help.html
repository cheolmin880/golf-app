🏌️ 6라운드 게임
18홀을 6홀씩 3라운드로 나눠 진행해요.
각 라운드마다 순위를 매기고 핸디를 조정해서
실력 차이가 있어도 재미있게 즐길 수 있어요!
⚙️ 게임 설정
1
플레이어 이름 입력 (최대 4명)
2
18홀 핸디캡 입력 (예: 핸디 12 → 18홀에서 12타 감점)
3
납부금액 설정 (1등/2등/3등/4등 금액)
4
버디벌금 설정 (버디/이글 시 모두가 내는 금액)
5
스타트홀 설정 (1번홀 또는 10번홀 등)
🏆 점수 계산 방법
이글 -2
버디 -1
파 0
보기 +1
더블 +2
트리 +3
각 홀의 오버/언더 타수를 입력해요.
파 기준으로 ±타수로 입력합니다.
🔄 라운드별 핸디 조정
6홀 라운드가 끝날 때마다 자동으로 핸디가 조정돼요.
1등 핸디 감소 (더 불리해짐)
꼴등 핸디 증가 (더 유리해짐)
→ 실력 차이를 줄여서 모두가 경쟁할 수 있어요!
💰 버디벌금
버디(-1) 또는 이글(-2)을 하면
나머지 플레이어 전원이 설정된 금액을 버디한 사람에게 내요.
예) 버디벌금 Rp10,000 × 3명 = Rp30,000 획득
💳 최종 정산
결과 탭에서 최종 순위와 정산금액 확인
버디벌금 + 라운드 납부금이 자동 계산돼요.
누가 누구에게 얼마 형식으로 표시!
📊 스트로크 게임
18홀 총 타수로 순위를 정하는 가장 기본적인 방식이에요.
핸디캡을 적용한 넷스코어로 공평하게 경쟁해요!
⚙️ 게임 설정
1
플레이어 이름 입력 (최대 4명)
2
핸디캡 입력 (18홀 기준)
3
단가 설정 (1타당 금액)
4
배반 설정 (꼴등 배수)
🧮 넷스코어 계산
넷스코어 = 총타수 - 핸디캡
예) 총타수 90, 핸디 18 → 넷스코어 72
💰 정산 방법
넷스코어 차이 × 단가로 계산해요.
꼴등은 배반(배수) 적용!
예) 단가 Rp5,000, 타수차 3타 → Rp15,000
📱 스코어 입력 방법
1
플레이어 이름 탭 → 해당 홀 탭
2
팝업에서 이글/버디/파/보기/더블+ 선택
3
입력 즉시 Firebase에 실시간 동기화
4
결과 탭에서 순위 및 정산 확인
📋 스코어카드
전통적인 골프 스코어카드 방식이에요.
전반(1~9홀) / 후반(10~18홀)로 나눠서
각 홀 타수를 직접 입력해요.
⚙️ 설정 방법
1
코스명 입력 (예: 자바베카)
2
스타트홀 설정 (1번홀 또는 10번홀 등)
3
홀별 파(Par) 설정 - 탭해서 P3/P4/P5 변경
4
플레이어 이름 · 핸디캡 입력
⛳ 파(Par) 설정 팁
홀별 파 설정은 1~18홀 전체 기준으로 입력해요.
스타트홀이 12번이면 → 12번홀부터 시작되지만
파 설정은 항상 실제 홀번호 기준으로 입력!
P3P4P5 탭할 때마다 순환해요.
📱 스코어 입력 방법
1
표에서 입력할 칸 탭
2
팝업에서 타수 선택
3
전반 완료 후 후반→ 버튼으로 이동
4
💾 저장 버튼으로 저장
🧮 넷스코어 자동 계산
그로스 = 실제 총타수
넷스코어 = 그로스 - 핸디캡
각 플레이어별로 자동 계산돼요
💡 알아두면 좋은 팁
📶 실시간 동기화
방장이 스코어를 입력하면 참가자 화면에 즉시 반영돼요.
● 연결됨 표시 확인!
⚠️ 카카오톡 브라우저는 동기화가 안 될 수 있어요!
우측 하단 ··· → 크롬으로 열기 를 선택하세요.
🔄 이전 방 이어하기
앱을 껐다가 다시 켜도 걱정 없어요!
▶️ 이전 방 이어하기 버튼이 자동으로 뜨고
탭하면 방금 전 게임으로 바로 복귀해요.
💾 자동 저장
스코어 입력 즉시 자동으로 저장돼요.
앱을 나가도 데이터가 사라지지 않아요.
🏠 홈 버튼을 누르면 저장 확인 팝업이 뜨는 경우도 있어요.
↩ 되돌리기
잘못 입력했을 때 ↩ 버튼을 탭하면
직전 입력으로 되돌릴 수 있어요.
📂 불러오기
이전에 저장한 게임 결과를
📂 불러오기 버튼으로 다시 볼 수 있어요.

새 라운딩을 시작할 때는
⚙️ 설정 → 🔄 초기화 버튼을 탭하세요.
새 방 코드가 생성되고 게임이 초기화돼요.
